Chapter 1: Sets

In this first chapter, we will learn about sets and how to represent them, what types of set are there, their properties, and the operations we can perform on them. To better understand and apply these ideas, we'll explore how to implement these concepts in Python to reinforce them. This section is particularly important, as it will be used in the following chapters.

1.1 What is a set?

A set is a collection of objects in common, each object is called element of the set. The sets are represented by capital letters and the elements are into curly braces {}, let's look at an examples:

Example

Let \(A,B,C\) be the following sets:

\[\textbf{A}= \{2,4,6,8\}\]

\[\textbf{B}= \{♣,♦,♥,♠\}\]

\[\textbf{C}= \{⚀,⚁,⚂,⚃,⚄,⚅\}\]
